## 📌 About Pandas

**Pandas** is a powerful Python library used for:

* Data cleaning 🧹
* Data manipulation 🔄
* Data analysis 📊
* Handling structured datasets (tables, CSVs, Excel, etc.)

It is widely used in **Data Science, Machine Learning, and Analytics**.
