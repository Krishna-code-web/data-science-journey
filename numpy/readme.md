## 📌 About NumPy

NumPy (Numerical Python) is the core library for numerical computing in Python.
It provides:

* High-performance multidimensional arrays
* Mathematical and statistical operations
* Foundation for libraries like Pandas, Matplotlib, Scikit-learn, TensorFlow
